Cloudiness


Cloudiness Description

There does not appear to be any correlation between city latitude and cloudiness. The cloudiness measurements for the cities included in the data set do seem to be clustered at certain percentages of cloudiness, including 0%, 20%, 40%, 75% and 100%. I would assume that these percentage measurements are likely some of the most common percentages used in measuring cloudiness. Beyond this visible clustering in relation to cloudiness, the values are spread across the range of latitude values present.
