Summary: Weather Data Visualizations

The visualizations on this page were generated as part of a Python API exercise for the Data Analytics and Visualization Bootcamp. Each of the visualizations compares a weather measurement of a given city to the latitude of the city and plots the results on a scatterplot.

By looking at each of these plots together and separately, it is clear that there are certain weather measurements that have a strong relationship with latitude (such as maximum temperature) and others in which the relationship is less clear (such as cloudiness and windspeed).

A more detailed description of each the plots are available on the individual plot pages. These pages can be accessed by using the dropdown menu in the navigation bar above. The comparisons page also contains all four visualizations displayed together. The data used to generate the visualizations are available on the data page.
